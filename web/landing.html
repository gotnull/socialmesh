Features

Everything You Need for Mesh Communication

Purpose-built for Meshtastic with no compromises. Every feature designed with input from the community.

Full Messaging

Direct messages, group channels, and broadcast support. Message history, delivery confirmations, and emoji reactions.

Interactive Map

Real-time node positions with offline map support. Track movement, view historical paths, and measure distances.

Telemetry Dashboard

Battery levels, signal strength, temperature, humidity, and air quality sensors. Historical charts and trend analysis.

Device Configuration

Complete radio settings, channel configuration, and module setup. No features locked behind a web interface.

Multiple Connections

Bluetooth LE, Wi-Fi, and USB serial support. Connect to multiple devices and switch seamlessly between them.

Custom Widgets

Build your own dashboard with drag-and-drop widgets. Charts, gauges, status displays, and quick action buttons.

Smart Notifications

Customizable alerts for messages, node events, and telemetry thresholds. Background operation with push notifications.

Network Analysis

Visualize mesh topology, trace routes between nodes, and analyze network health. MQTT bridge support included.

Privacy First

No accounts required. No cloud dependencies. All data stays on your device. Open protocol, no vendor lock-in.

Capabilities

Not Just a Companion App

Socialmesh is a standalone Meshtastic client with full feature parity. Everything you can do with the official tools, you can do here.

  • Complete Channel Management

    Create, edit, share, and import channels with full encryption support

  • All Radio Settings

    Region, modem preset, hop limit, and advanced LoRa parameters

  • Module Configuration

    Position, telemetry, range test, store & forward, and external notifications

  • Firmware Updates

    OTA updates for supported devices with version management

  • Debug Tools

    Raw packet inspection, protocol debugging, and diagnostic logs
